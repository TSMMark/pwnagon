# pwnagon

Git clone this repo and `cd` to the directory.

```shell
vagrant up
```

```shell
vagrant ssh
```

```shell
cd pwnagon
```

```shell
bundle install
```

```shell
rake db:migrate
```

```shell
rake db:seed
```

```shell
rails s -b 0.0.0.0
```

Visit `localhost:3000` in your favorite browser.
